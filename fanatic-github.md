#### 4. **WindowManager** 🔄 Планируется переписывание
- **Язык**: Python (текущая версия), **План**: Go
- **Лицензия**: MIT
- **Описание**: Легковесная утилита для управления окнами Windows из командной строки.
- **Примеры использования**:
  ```bash
  windowmanager -l                    # Список окон
  windowmanager -e -t "Notepad"      # Проверить, запущен ли Notepad
  windowmanager -m -t "Calculator"   # Свернуть Calculator
  windowmanager -f -p "chrome"       # Активировать окно Chrome
  ```
- **Статус**: ✅ Работает, но автор планирует переписать с нуля на **Go** для лучшей производительности.
- **Ссылка**: [github.com/FanaticExplorer/WindowManager](https://github.com/FanaticExplorer/WindowManager)

### 📊 Data Engineering / Аналитика

#### 11. **uni-nfl-fantasy-dwh** 🎓 Учебный проект по Data Warehouse
- **Языки**: SQL (Snowflake)
- **Описание**: Реализация ELT-процесса и создание Data Warehouse по методологии Kimball для датасета **NFL Fantasy Football** из Snowflake Marketplace.
- **Архитектура**:
  - **Схема**: Star Schema
  - **Факты**: `fact_player_game_stats`
  - **Измерения**: `dim_player` (SCD Type 2), `dim_team`, `dim_game`, `dim_date`, `dim_play_type`
- **Аналитические возможности**:
  - Ранжирование игроков по неделям
  - Сравнение домашний/выездной игры
  - Анализ влияния погоды и типа стадиона
  - Выявление "прорвавшихся" игроков (breakout stars)
  - Расчёт fantasy-очков по формулам
- **Визуализации**: 7 SQL-запросов для аналитических дашбордов
- **Статус**: ✅ Завершённый учебный проект, оцененный и принятый университетом
